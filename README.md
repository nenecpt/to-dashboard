# TO 線控管理中心

> 行健旅遊 UNOTOUR · Tour Operator 個人線控管理儀表板

## 功能

- 📊 **總覽儀表板** — 20 條線一覽無遺的狀態卡片牆
- 🗺️ **線路管理** — 每條線的詳細資訊 CRUD
- ✅ **待辦 + 截止日倒數** — 跨線路聚合 + 顏色警示
- 💰 **團費收支** — 每團損益即時計算
- 🔍 全域搜尋
- 📤📥 資料匯出/匯入（JSON）
- 📱 全裝置響應式設計

## 使用方式

直接用瀏覽器開啟 `index.html` 即可。首次開啟會自動載入 3 筆示範資料。

## 資料儲存

所有資料存於瀏覽器 localStorage，建議定期使用「匯出」功能備份。
